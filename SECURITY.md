# Security Policy

## Supported Versions

Please use the latest version for the best security and performance.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please **DO NOT** open a public issue. Instead, please report it privately to the repository maintainers.

1. Email the project maintainer with a description of the issue.
2. Provide details on how to reproduce the vulnerability.
3. Allow the maintainers time to investigate and patch the issue before publicly disclosing it.

We take all security issues seriously and will respond as quickly as possible.
